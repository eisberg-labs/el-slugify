# EL-SLUGIFY

> URL slug generator utility. Slugs are generated efficiently, fast, they are transliterated and sanitized for use in URLs.

What makes a good slug?

- transliterated
- lowercased
- sanitized

You can use el_slugify in your rust and node projects.
